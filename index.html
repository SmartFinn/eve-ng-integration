UNetLab-X-Integration

a tool which helps UNetLab to integrate with Ubuntu/Debian

View on GitHub Download .zip Download .tar.gz

Description

UNetLab-X-Integration is the equivalent of Windows integration pack for Ubuntu/Debian.

After install the package you can open URLs like the following:

  • telnet://
  • capture://
  • docker://
  • vnc:// (via Vinagre)
  • rdp:// (via Vinagre)
  • ssh:// (via Vinagre)

Tested on Ubuntu 14.04 with Google Chrome and Firefox.
